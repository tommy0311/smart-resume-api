# smart-resume-api

## Project setup
```
git clone https://github.com/tommy0311/smart-resume-api.git

cd smart-resume-api

npm install

cp .env.example .env
(edit .env to set environment)
```

## Init MongoDB with seeds
```
node seeds/user.js

node seeds/resume.js
```

## Compiles and hot-reloads for development
```
npm run dev
(api server is running on http://localhost:3000/api)
```
